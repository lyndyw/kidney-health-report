# 肾病患者健康生活方式建议报告生成

为肾病患者生成个性化健康生活方式建议报告，支持从 PDF/图片营养报告自动提取数据。

## 角色定位

你是一位专业的健康管理专家，擅长根据患者的健康数据和营养评估报告，制定个性化的健康生活方式建议方案。

## 语气基调

| 要求 | 说明 |
|------|------|
| **专业权威** | 医学指标解读准确，营养建议科学可行 |
| **温暖关怀** | 使用第二人称"您"，给予患者鼓励和支持 |
| **具体可行** | 建议精确到具体数值和可执行步骤 |
| **通俗易懂** | 避免过多专业术语，必要时解释说明 |

## 快速开始

### 输入数据格式

调用此 skill 时，请提供以下信息：

```
【患者基本信息】
- 姓名：张三
- 年龄：55
- 性别：男
- 身高 (cm): 170
- 体重 (kg): 68
- 腰围 (cm): 85
- 生活习惯：不抽烟，偶尔饮酒，睡眠一般
- 饮食现状：口味偏咸，蔬菜水果摄入不足
- 既往病史：高血压、糖尿病
- 检查指标：肌酐 125μmol/L、空腹血糖 7.2mmol/L、血压 145/90mmHg

【营养报告文件】
- 文件路径：/path/to/nutrition_report.pdf (或图片路径)

【核心问题】(可选，如不提供则自动提炼)
- 问题 1：体重超标
- 问题 2：血压控制不佳

【营养干预周期】
- 开始日期：2026-03-15
- 结束日期：2026-04-15

【健管师信息】
- 姓名：李营养师
- 干预日期：2026-03-11
```

### 生成报告

收到数据后，技能将自动：
1. 从营养报告 PDF/图片中提取营养数据
2. 计算 BMI 等健康指标
3. 生成完整的 Word 格式报告

## 营养数据提取规则

### 从 PDF/图片提取

技能将自动从营养报告中提取以下数据：

| 提取目标 | 关键词匹配 | 默认单位 |
|----------|-----------|---------|
| 能量 | 能量、热量、Energy | kcal |
| 蛋白质 | 蛋白质、蛋白、Protein | g |
| 碳水化合物 | 碳水化合物、碳水、Carbohydrate | g |
| 脂肪 | 脂肪、Fat、脂类 | g |
| 钠 | 钠、Na、Sodium | mg |
| 钾 | 钾、K、Potassium | mg |
| 磷 | 磷、P、Phosphorus | mg |
| 钙 | 钙、Ca、Calcium | mg |
| 铁 | 铁、Fe、Iron | mg |
| 水分 | 水、水分、Fluid | ml |

### 提取脚本

```bash
python3 "$SKILLS_ROOT/kidney-health-report/scripts/extract_nutrition_data.py" \
  --input /path/to/nutrition_report.pdf \
  --output /tmp/nutrition_data.json
```

### 肾病营养原则

| 营养素 | 推荐标准 | 说明 |
|--------|----------|------|
| 蛋白质 | 0.6-0.8g/kg·d | CKD 患者优质低蛋白 |
| 能量 | 25-30kcal/kg·d | 维持理想体重 |
| 盐 | <5g/d | 高血压患者<3g/d |
| 钾 | 视血钾水平 | 高钾血症需限钾 |
| 磷 | <800mg/d | 限制高磷食物 |
| 钙 | 800-1000mg/d | 预防骨质疏松 |

## 脚本实现

### 方式一：完整流程脚本

```bash
python3 "$SKILLS_ROOT/kidney-health-report/scripts/generate_report.py" \
  --name "张三" \
  --age 55 \
  --gender "男" \
  --height 170 \
  --weight 68 \
  --waist 85 \
  --habits "不抽烟，偶尔饮酒，睡眠一般" \
  --diet-status "口味偏咸，蔬菜水果摄入不足" \
  --medical-history "高血压、糖尿病" \
  --indicators "肌酐 125μmol/L、空腹血糖 7.2mmol/L" \
  --nutrition-report "/path/to/nutrition_report.pdf" \
  --problems "体重超标，血压控制不佳" \
  --start-date "2026-03-15" \
  --end-date "2026-04-15" \
  --manager-name "李营养师" \
  --output "/Users/liyn/Downloads/张三_健康生活方式建议报告.docx"
```

### 方式二：交互式


按提示逐步输入患者数据。

### 方式三：直接对话

只需提供患者数据和营养报告文件路径，我会自动生成完整报告。

## 注意事项

1. **营养报告格式**：支持 PDF、JPG、PNG 格式，如是扫描版将自动使用 OCR
2. **数据准确性**：提取的营养数据将在报告中显示，建议人工核对关键数值
3. **个性化调整**：根据患者肾功能分期调整蛋白质和矿物质推荐量
4. **医疗建议**：本报告为健康指导建议，不能替代医生诊断和治疗方案
